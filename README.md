# From Code to Charts: Create DataWrapper Graphics Straight from R
Supercharge your visualization workflow by using the DataWrapper API to create powerful graphics directly from R. Clean, analyze and visualize data without exporting or switching between applications. Build charts and maps, create templates, automate production tasks and produce dozens (or hundreds) of charts with just a few lines of code. This session is good for journalists with some knowledge of DataWrapper and R.
